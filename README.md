## 🚀 Projetos em Destaque

### 🔍 Search+ — Busca Semântica de Imagens com IA
> *O projeto mais ambicioso que já construí*

**Search+** é um sistema inteligente de busca de imagens por linguagem natural. Você faz upload de uma imagem, a IA gera automaticamente uma descrição detalhada do conteúdo, e essa descrição é armazenada no banco de dados. Depois, basta digitar o que você quer encontrar em texto — e o sistema recupera as imagens correspondentes.

**Como funciona:**
```
[ Upload de Imagem ]
        ↓
[ IA gera descrição automática ]
        ↓
[ Armazena descrição no banco ]
        ↓
[ Usuário busca por texto ]
        ↓
[ Sistema retorna imagens correspondentes ]
```

**Stack & conceitos envolvidos:**
- 🤖 Integração com IA para geração de descrições (visão computacional)
- 🗄️ Banco de dados para armazenamento das descrições
- 🔎 Busca semântica por linguagem natural
- 🌐 Interface web para upload e consulta

> Este projeto nasceu como trabalho escolar e evoluiu para o meu principal portfólio técnico — uma aplicação real que combina IA, banco de dados e desenvolvimento web do zero.
